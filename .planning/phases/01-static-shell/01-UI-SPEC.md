# Phase 1 — UI Design Contract

> Visual and interaction contract for the static shell. Generated by gsd-ui-researcher, verified by gsd-ui-checker.
> This phase delivers layout structure and placeholder content only — no live data, no interactions.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | none |
| Preset | not applicable |
| Component library | none — pure HTML/CSS |
| Icon library | none |
| Font | system-ui, -apple-system, sans-serif (system stack, no external load) |

All tokens are declared as CSS custom properties in a `<style>` block inside the single `.html` file. No external stylesheet. No CDN. No import.

## Typography

All font sizes use `px` units. Line heights are unitless ratios.

| Role | CSS var | Size | Weight | Line Height | Usage |
|------|---------|------|--------|-------------|-------|
| Body | --text-body | 14px | 400 | 1.5 | Stat values, station IDs, general prose |
| Label | --text-label | 12px | 400 | 1.4 | Stat row labels ("Gage Height", "Discharge", etc.) |
| Heading | --text-heading | 18px | 600 | 1.2 | Station card titles (station name / ID) |
| Display | --text-display | 24px | 600 | 1.2 | Dam section heading ("Buford Dam Release Schedule") |

Font stack (applied to `body`): `system-ui, -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif`

No web fonts. No @font-face. No Google Fonts import.

## Color

Dark theme. All colors declared as CSS custom properties.

| Role | CSS var | Hex | Usage |
|------|---------|-----|-------|
| Dominant (60%) | --color-bg | #0f1117 | Page background |
| Secondary (30%) | --color-surface | #1c1f26 | Card backgrounds, dam section background |
| Surface-alt | --color-surface-alt | #252830 | Card header band (station name row), subtle dividers |
| Accent (10%) | --color-accent | #3b82f6 | Dam section heading underline only |
| Border | --color-border | #2e3138 | Card border, section border |
| Text-primary | --color-text | #e2e8f0 | All heading and value text |
| Text-muted | --color-text-muted | #8b92a5 | Stat labels, station IDs, placeholder dashes |
| Destructive | --color-destructive | #ef4444 | Not used in Phase 1 |

Accent reserved for: the decorative underline/left-border on the dam section heading. Not applied to cards, links, or any other element in Phase 1.

60/30/10 breakdown:
- 60%: `--color-bg` fills the full page viewport
- 30%: `--color-surface` fills all 9 station cards and the dam section panel
- 10%: `--color-accent` used only as a 3px left border on the dam section `<h2>`

## Layout Contract

### Page Structure

```
<body>                          — bg: --color-bg
  <header>                      — page title "HoochSnapshot", text-muted, text-label size
  <main>
    <section id="dam">          — dam placeholder panel, full width
    <section id="stations">     — card grid
      <div class="card"> × 9   — one per station, fixed order
```

### Dam Section

- Full-width panel, `padding: var(--space-2xl) var(--space-xl)`
- Background: `--color-surface`
- Border-bottom: `1px solid var(--color-border)`
- `<h2>` "Buford Dam Release Schedule" at display size, with `border-left: 3px solid var(--color-accent)` and `padding-left: var(--space-sm)`
- Body text placeholder: "Dam release schedule will appear here." in `--color-text-muted`

### Station Card Grid

- CSS Grid: `grid-template-columns: repeat(auto-fill, minmax(280px, 1fr))`
- Gap: `var(--space-lg)` (24px)
- Padding around grid: `var(--space-xl)` (32px) on left/right, `var(--space-lg)` (24px) top/bottom

### Station Card

- Background: `--color-surface`
- Border: `1px solid var(--color-border)`
- Border-radius: `8px`
- No box-shadow in Phase 1
- Internal padding: `var(--space-md)` (16px)
- Card header band (station name + ID row):
  - Background: `--color-surface-alt`
  - Padding: `var(--space-sm) var(--space-md)` (8px 16px)
  - Margin: `-var(--space-md) -var(--space-md) var(--space-md) -var(--space-md)` (bleed to card edges)
  - Border-radius on top corners only: `8px 8px 0 0`
- Stat rows: `display: flex; justify-content: space-between; padding: var(--space-xs) 0`
- Divider between stat rows: `border-bottom: 1px solid var(--color-border)` on all rows except the last
- Station name: heading size (18px), weight 600, `--color-text`
- Station ID: label size (12px), weight 400, `--color-text-muted`, shown below station name in header band
- Stat label: label size (12px), weight 400, `--color-text-muted`
- Stat value: body size (14px), weight 400, `--color-text`

### Station Order (fixed, REQUIREMENTS LAYOUT-02)

Cards render in this exact DOM order:

1. 023344030
2. 02335000
3. 02335450
4. 02335815
5. 02335880
6. 02336000
7. 02337170

9. 02338500

### Stat Rows Per Card

Turbidity rows appear only on cards for stations 02335000, 02335880, 02336000. All other cards have exactly 3 stat rows. Cards with turbidity have 4 stat rows.

| Stat | Label | Placeholder value | Unit shown |
|------|-------|-------------------|------------|
| Gage Height | Gage Height | — | ft |
| Discharge | Discharge | — | cfs |
| Temperature | Temperature | — | °F |
| Turbidity (3 stations only) | Turbidity | — | FNU |

In Phase 1, all values are the en-dash character (`—`) in `--color-text-muted`. Units are shown inline after the dash: `— ft`.

## Interaction Contract

Phase 1 has zero interactive elements. No click handlers. No hover states. No JavaScript. The file is pure static HTML + CSS.

- Cards are not clickable in Phase 1 (clickability is Phase 2, CARD-03)
- No refresh button (Phase 2 / ENH-01)
- No loading spinners (no data fetching in Phase 1)
- Cursor: default on all elements (no pointer cursor)
